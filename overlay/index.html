API

Class

Overlay

继承自Component。

Options

new Overlay()

参数类型默认值数据流向描述
dataobject绑定属性
data.openbooleanfalseouter <=> inner当前为展开/收起状态
data.disabledbooleanfalseouter => inner是否禁用
data.visiblebooleantrueouter => inner是否显示
data.classstring''outer => inner补充class

Methods

overlay.toggle(open)

展开/收起

参数类型默认值描述
openboolean展开/收起状态。如果无此参数，则在两种状态之间切换。
返回值类型描述
无返回值

Events

overlay.$on('toggle')

展开/收起时触发

属性类型描述
senderobject事件发送对象
openobject展开/收起状态
